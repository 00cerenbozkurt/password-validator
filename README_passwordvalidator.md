# ✨ Password Validator ✨

A soft little guardian for your passwords 🌙\
This Python program checks how strong your password is and helps you
create safer ones.

------------------------------------------------------------------------

## 🧚 Features

-   🌿 Checks password length
-   🌸 Detects uppercase & lowercase letters
-   🔮 Finds numbers and symbols
-   🪄 Gives a strength score
-   ♻️ Lets you retry until it's strong

------------------------------------------------------------------------

## 🌙 How It Works

You enter a password, and the program gently inspects it:

-   Too short? It warns you 🥀\
-   Missing elements? It guides you ✨\
-   Strong enough? It approves your magic 🔐

------------------------------------------------------------------------

## 🌟 Example

    Enter password: Abc123!
    Strong password (5/5)

------------------------------------------------------------------------

## 🍃 Run Locally

``` bash
python password_validator.py
```
